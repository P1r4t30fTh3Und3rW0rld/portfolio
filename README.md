# Portfolio

Personal portfolio website built with React, TypeScript, and Tailwind CSS.
heavily inspired by nexxel.dev

## Getting Started


```bash
# Go into backend/
cd .\backend\
# Install dependencies
npm install

# Start development server
npm run dev
```

```bash
# Go into frontend/
cd .\frontend\
# Install dependencies
npm install

# Start development server
npm run dev
```

now visit: http://localhost:8080/

## Build

```bash
npm run build
```
